AI-Informed Searches Assignment
This project implements a grid-based pathfinding system using **Greedy Best-First Search** and **A\* Search algorithms**.  
The program is built with **Python** and **Pygame** and visualizes how these algorithms explore a grid to find a path from a Start node to a Goal node.

 Algorithms Implemented
- Greedy Best-First Search
- A* Search

Requirements
Make sure Python is installed. Then install the required dependency:

pip install pygame


How to Run
1. Clone the repository
  git clone https://github.com/aanisbilal/AI-Informed-searches-Assignment.git
2. Navigate into the folder
  cd AI-Informed-searches-Assignment
3. Run the program
  python main.py

Technologies Used
- Python
- Pygame
